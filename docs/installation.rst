Installation
============

Requirements
------------

:mod:`ncplib` supports Python 3.4.2 and above. It has no other dependencies.


Installing
----------

It's recommended to install :mod:`ncplib` in a virtual environment using :mod:`venv`.

Install :mod:`ncplib` using `pip`_.

.. code:: bash

    pip install ncplib


Upgrading
---------

Upgrade :mod:`ncplib` using `pip`_:

.. code:: bash

    pip install --upgrade ncplib

.. important::

    Check the :doc:`changelog` before upgrading.
