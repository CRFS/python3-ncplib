Contributing
============

Bug reports, bug fixes, and new features are always welcome. Please raise issues on `GitHub`_, and submit pull requests for any new code.


Testing
-------

It's recommended to test :mod:`ncplib` in a virtual environment using :mod:`venv`.

Run the test suite:

.. code:: bash

    pip install -e .
    python -m unittest discover tests
