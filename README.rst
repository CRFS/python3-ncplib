ncplib
======

`CRFS <http://www.crfs.com/>`_ NCP library for Python 3.


Features
--------

- NCP client library.
- Asynchronous by default, using `asyncio <https://docs.python.org/3.4/library/asyncio.html>`_.
- Works in Python 3!


Installation
------------

1. Install using ``pip install ncplib``.
